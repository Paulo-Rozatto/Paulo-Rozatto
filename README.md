## 🔬 Research & Publications

### 1. Soft Routing-Inspired Specialization for Efficient Satellite Image Time Series Segmentation

> *Published in ICCSA 2026*

As the main focus of my current research, this paper introduces a soft routing-inspired specialization into the Exchanger architecture. By adapting soft Mixture-of-Experts (MoE) principles, we successfully maintain state-of-the-art performance while significantly reducing computational resources, cutting FLOPs by up to 30% and lowering latency, for large-scale satellite image time series (SITS) segmentation.
- [🔗 Code Repository](https://github.com/Paulo-Rozatto/exchanger-soft-routing)

### 3. A Computer Vision-based Architecture for Remote Physical Rehabilitation

> *Published in Journal of Mobile Multimedia (2024)*

We proposed a software architecture that utilizes computer vision to assist in the remote diagnosis and treatment of physical rehabilitation patients. The system tracks prescribed exercises and provides visual guidance to ensure patients execute them properly from home.
- [📄 Read the Paper](https://doi.org/10.13052/jmm1550-4646.2045)
